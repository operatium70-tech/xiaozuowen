# 世界规则优先级

## 优先级顺序

1. 已确认世界底层规则。
2. 已写入正文且不可轻易推翻的事实。
3. 已写入 `1_世界观/` 的设定。
4. 已写入 `3_剧情大纲/` 的规划。
5. 已写入 `7_长期记忆/` 的当前状态。
6. 临时创作建议。

## 冲突处理

当新想法与旧设定冲突时，优先保护已发布正文。

如果必须修改旧设定，需要记录：

- 冲突来源
- 修改原因
- 影响章节
- 需要同步更新的文件

## 新规则进入流程

重大世界规则不得直接进入正文。

推荐流程：

1. 写入 `3_剧情大纲/未回收设定.md`。
2. 判断是否影响主线。
3. 补充到 `1_世界观/` 对应文件。
4. 更新 `7_长期记忆/当前世界变化.md`。
5. 再在正文中渐进释放。
