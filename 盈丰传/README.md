# 盈丰传

## 实现原理

本项目是《盈丰传》的 Obsidian 小说工程。它参考 `operatium70-tech/xiaozuowen2` 的工程化结构，但不继承其故事设定、人物、世界观或正文内容。

核心原则是：作者定规则、世界观、人设和节奏；AI 只按规则执行，不自创设定。

## 如何使用

打开项目后先读 [[总控制大屏]]，再按总控制大屏进入系统规则、长期记忆、剧情大纲、世界观、人物设定、文风控制和技能规则。

## 根目录结构

```text
Obsidian工作台/     插件工作流、模板和工程说明
系统规则/         最高优先级规则、正史判定和冲突处理
世界观/           世界设定、元素熵变、境界、资源规则
人物设定/         人物档案、人物索引和关系入口
剧情大纲/         定稿大纲、复盘、时间线和后续细纲
正文源码/         未来唯一正文源，当前禁止写正文
文风控制/         小说定位、文风、节奏和表达边界
参考资料/         非正史参考、阶段存档、命名和现实资料
长期记忆/         当前状态、伏笔、时间线、人物关系和统计
技能规则/         AI 专项执行规则
发布版本/         未来发布版、平台版和全文整合版
素材资源/        封面、人物图、地图、提示词和版权来源
tools/              只读检查、导出和统计脚本入口
```
