# 伏笔回收总表

## 实现原理

本文件管理候选伏笔的长期回收路线。当前尚未写正文，因此所有条目均为候选，不进入 [[剧情大纲/定稿/全局/伏笔系统|伏笔系统]] 正式登记。

## 如何使用

拆章节前先看 [[剧情大纲/定稿/全局/伏笔候选投放表|伏笔候选投放表]] 决定是否投放；拆卷纲和阶段终点时看本表决定如何推进、变形和回收。

## 状态规则

| 状态 | 含义 | 可否进入伏笔系统 |
| --- | --- | --- |
| 候选 | 只在大纲中存在，尚未写入正文 | 不可 |
| 已投放 | 正文中出现，但读者只看见表层用途 | 可 |
| 已推进 | 正文中出现第二次，意义发生变化 | 可 |
| 待回收 | 已经明确进入主线，但尚未解决 | 可 |
| 已回收 | 正文中完成解释、选择或代价支付 | 可 |

## 使用规则

1. 每章最多投放一个主伏笔。
2. 每个伏笔必须先服务当前剧情，再服务长线。
3. 感情伏笔不能只为虐恋，必须服务人生认知。
4. 技术伏笔不能变成神器解释。
5. 终局回收必须能追溯到低阶影子。
