# 师友同伴与主角团线

## 实现原理

本线管理长期同伴、导师、观察者、竞争者和执行伙伴。它的目的不是凑热闹，而是让赵盈丰的成长有真实的人际成本和组织成本。

## 如何使用

新增或使用任何重要配角前，先确认此人服务哪条线、照出赵盈丰哪项缺陷、带来什么成本、最终是否独立完成自身成长。

## 核心命题

真正的同伴不是永远同意你，而是在你正确但不成熟、强大但误判、想救人但承担不起时，逼你把个人判断变成可共同承担的行动。

## 阶段推进表

| 阶段 | 同伴功能 | 赵盈丰必须付出的成本 |
| --- | --- | --- |
| 第一阶段白丁境 | 从独自证明自己，到被迫与队友协作 | 改表达、分功劳、承担误会 |
| 第二阶段炼气境 | 临时小队和组织任务让判断进入流程 | 写报告、服从纪律、接受复核 |
| 第三阶段筑基境 | 项目团队检验他的局部规则 | 分权、授权、失败责任 |
| 第四阶段万象境 | 异星驻守团队检验适配能力 | 尊重陌生文化和生存节奏 |
| 第五阶段大师境 | 工程团队把个人方法转成技术路线 | 承认别人比自己更专业 |
| 第六阶段大宗师境 | 标准制定伙伴和反对者共同塑造公共规则 | 妥协、谈判、透明化代价 |
| 第七阶段至尊境 | 文明盟友检验救援边界 | 不能替别人做所有选择 |
| 第八阶段永恒境 | 少数同行者面对静恒诱惑 | 接受有人选择不同道路 |
| 第九阶段帝境 | 道路继承者让盈丰路不依赖赵盈丰个人 | 放手，让文明自己承担 |

## 使用规则

1. 不新增只负责称赞、献祭或送资源的同伴。
2. 同伴必须有独立欲望和独立恐惧。
3. 同伴不能永远正确，也不能永远拖后腿。
4. 导师不能替赵盈丰解决核心选择。
5. 观察者的评价必须带来机会，也带来约束。
6. 周燕的终局位置是共担者，不是附属者。
