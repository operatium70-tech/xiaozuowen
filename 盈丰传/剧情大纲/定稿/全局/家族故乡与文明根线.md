# 家族故乡与文明根线

## 实现原理

本线保证赵盈丰离开赵王星后，故事不会失去根。[[赵王星]]、[[赵氏家族]]、学院和早期关系必须持续成为他理解文明、阶层、家庭和责任的参照。

## 如何使用

每次地图扩大后，都要安排一个“故乡回流点”：赵王星如何被更大规则影响，赵盈丰如何重新理解早期人和事。

## 核心命题

人走得越远，越要明白自己从哪里来。故乡不是束缚，也不是滤镜，而是一个人判断文明代价时最初的尺度。

## 阶段推进表

| 阶段 | 家族与故乡功能 | 赵盈丰理解变化 |
| --- | --- | --- |
| 第一阶段白丁境 | 赵氏旁支位置、学院竞争、赵倩倩稳定上升 | 第一次看见平台差异 |
| 第二阶段炼气境 | 家族开始注意，但不真正信任 | 明白被看见不等于被接纳 |
| 第三阶段筑基境 | 家族重评和赵倩倩成熟分离完成 | 明白安稳人生也有其合理性 |
| 第四阶段万象境 | 离开赵王星，异星规则反照母星局限 | 明白故乡规则不是宇宙通用规则 |
| 第五阶段大师境 | 星系工程影响赵王星资源路线 | 明白技术路线会改写故乡命运 |
| 第六阶段大宗师境 | 星域标准可能压迫母星和弱小文明 | 明白标准不是中立的 |
| 第七阶段至尊境 | 文明尺度选择逼近故乡利益 | 明白爱故乡不能牺牲别人故乡 |
| 第八阶段永恒境 | 赵王星成为理解“变化与延续”的样本 | 明白故乡也必须变化才能活 |
| 第九阶段帝境 | 终局回望赵王星和早期人生 | 明白道路要让普通故乡也能承担 |

## 关键回流点

1. 赵氏旧评价：从“旁支无大能”到“旁支方法有公共价值”。
2. 赵倩倩人生：从少年吸引到成熟分离，再到稳定人生的尊重。
3. 学院比赛档案：低阶误差成为高阶元素生态问题的早期证据。
4. 赵王星资源路线：从本地材料不平等，变成星系技术标准样本。
5. 故乡普通人：终局不是强者长生，而是普通文明有可承担的局部有序方案。

## 使用规则

1. 家族不能只当压迫工具，也不能突然变成无条件后盾。
2. 故乡不能只在开头出现，后期必须有现实影响。
3. 赵盈丰不能靠否定故乡完成成长。
4. 赵倩倩线必须保持有缘无分，不改成回头复合。
5. 周燕最终家庭线应与共同道路相连，不写成退隐奖励。
