# 道心主题与读者教育线

## 实现原理

本线管理每个阶段要让读者悟到的处世原则、人际原则和修行原则。它不写说教正文，只规定主题落点。

## 如何使用

拆阶段和关系线时，必须写清“赵盈丰悟到什么、读者从关系或事件里看见什么、这个道理如何进入后续选择”。

## 核心命题

《盈丰传》的道心不是口号，而是在一次次关系破裂、组织误判、技术失败、阶层碰壁和文明代价中形成的稳定判断。

| 阶段 | 赵盈丰悟到 | 读者教育原则 |
| --- | --- | --- |

| 第二阶段炼气境 | 情绪不能替代证据 | 想改变别人，先让判断可复核 |
| 第三阶段筑基境 | 爱和人生路线不能互相拖拽 | 尊重对方的安稳，也是成熟 |

| 第五阶段大师境 | 个人聪明必须让位于可复制协作 | 长久关系依靠共同劳动，不靠新鲜感 |
| 第六阶段大宗师境 | 标准不是中立的，必须公开代价 | 公平不是口号，是谁承担成本的问题 |

## 处世原则池

1. 人与人相处，不能只问自己是否真诚，也要问对方是否承受得起。
2. 正确如果不能被理解、复核和承担，就还不是公共答案。
3. 爱有短保质期，稳定关系需要共同秩序维护。
4. 尊重不是放弃，而是不把自己的道路强加给别人。
5. 门当户对不是势利，而是生活节奏、资源承受力和责任结构的匹配。
6. 强者的善意如果不公开代价，可能变成另一种压迫。
7. 真正的成长不是无人反对，而是能在反对中修正自己。

## 使用规则

1. 道理必须从事件和选择中长出来，不写成作者直接训话。
2. 每段感情都要让女方也完成自己的人生，不做男主悟道工具。
3. 高阶主题必须能回扣低阶生活经验。
4. 终局主题必须回到普通人、故乡和长期关系。
