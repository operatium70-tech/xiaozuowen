# 势力组织与公共规则线

## 实现原理

本线管理学院、家族、商会、军方、地方机构、跨星组织、星域联盟和高阶文明之间的规则关系。

## 如何使用

每出现一个组织，必须写清它的利益、规则、评价标准、能给赵盈丰什么、会限制赵盈丰什么。

## 核心命题

组织不是背景板。真正的公共规则既保护人，也筛选人；既能让底层上升，也能把底层的代价藏起来。

## 组织层级表

| 层级 | 组织类型 | 功能 | 主要矛盾 |
| --- | --- | --- | --- |
| 学院层 | 学院、赛务、导师、训练体系 | 提供公开选拔和早期评价 | 公平口径与资源差异 |
| 家族层 | 赵氏主脉、嫡系、旁支、分部 | 提供血缘网络和身份压力 | 传统评价与新方法价值 |
| 母星层 | 商会、军方、地方任务机构 | 提供真实任务和公共责任 | 成果归属、组织纪律、商业包装 |
| 跨星层 | 外派机构、异星驻守点、航行系统 | 打开多元素生态 | 母星标准与异星实际冲突 |
| 星系层 | 工程集团、资源网络、航行联盟 | 推动技术路线复制 | 私有收益与公共标准 |
| 星域层 | 联盟、审议会、标准委员会 | 制定大范围公共规则 | 强组织制定标准、弱文明承担成本 |
| 星河层 | 高阶文明联合体 | 处理文明尺度危机 | 稳定与自由、救援与干涉 |
| 终局层 | 静恒体系和盈丰路支持者 | 争夺永恒方案 | 冻结稳定与迭代稳定 |

## 阶段推进表

| 阶段 | 组织功能 | 赵盈丰的关系变化 |
| --- | --- | --- |
| 第一阶段白丁境 | 学院和比赛提供低位入口 | 从边缘学生变成争议替补 |
| 第二阶段炼气境 | 军方和地方任务检验纪律 | 从被观察者变成低位任务参与者 |
| 第三阶段筑基境 | 家族和项目机构重新评估 | 从旁支青年变成可授权对象 |
| 第四阶段万象境 | 跨星驻守机构限制外来者 | 从母星人变成跨星行动者 |
| 第五阶段大师境 | 工程组织争夺技术路线 | 从执行者变成路线持有人 |
| 第六阶段大宗师境 | 星域组织争夺标准 | 从路线持有人变成规则参与者 |
| 第七阶段至尊境 | 星河组织处理文明危机 | 从规则参与者变成代价承担者 |
| 第八阶段永恒境 | 顶层组织诱导冻结方案 | 从代价承担者变成道路选择者 |
| 第九阶段帝境 | 文明自愿选择道路 | 从组织参与者变成路的留下者 |

## 使用规则

1. 组织不能无缘无故赏识赵盈丰。
2. 组织也不能无缘无故打压赵盈丰。
3. 每次组织给机会，都必须带条件。
4. 每次组织制造压力，都必须有自洽利益。
5. 最终公共规则必须公开代价，不能靠圣人意志维持。
