# 身份阶层与社会承认线

## 实现原理

本线管理赵盈丰从旁支少年到宇宙规则开路者的社会位置变化。它解决一个关键问题：主角为什么有资格进入下一层舞台。

## 如何使用

每阶段结尾必须写清赵盈丰获得了什么新身份、谁承认了他、承认依据是什么、承认后带来什么新压力。

## 核心命题

真正的上升不是别人终于看见你厉害，而是你做出的公共贡献足够明确，组织不得不重新分配给你责任和入口。

## 阶层压力来源

1. 家族出身：[[赵氏家族]] 的主脉、嫡系、旁支差异。
2. 教育平台：[[星球教育制度]] 和 [[学院生态与筛选机制]] 的筛选差异。
3. 资源渠道：旧护具、冷门材料、检测设备和样本库的不平等。
4. 组织信用：军方、商会、学院和家族对同一成果的评价不同。
5. 文明位置：强文明有标准制定权，弱文明承担更多外部代价。

## 使用规则

1. 每次身份跃迁都必须有公开事件支撑。
2. 每次承认都不能是全员服气，而是“不得不重新评估”。
3. 每次上升都要带来更重责任，不能只带来资源。
4. 赵盈丰不能靠隐藏血统或神秘靠山解决阶层问题。
5. 阶层批判要落到制度、成本和选择，不写成单纯仇富或打脸。
