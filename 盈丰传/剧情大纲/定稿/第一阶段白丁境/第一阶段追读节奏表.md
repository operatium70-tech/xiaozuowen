# 第一阶段追读节奏表

## 实现原理

本文件补足 [[剧情大纲/定稿/第一阶段白丁境/第一阶段逐章场景卡|第一阶段逐章场景卡]] 没有单独管理读者情绪曲线的问题。它不改变六十章事件，只规定每个单元应给读者的爽点、压迫、疑问和章末期待。

## 如何使用

写第一阶段正文前，先读 [[剧情大纲/定稿/第一阶段白丁境/第一阶段完整卷纲|第一阶段完整卷纲]]、[[剧情大纲/定稿/第一阶段白丁境/第一阶段逐章场景卡|第一阶段逐章场景卡]]，再用本表检查追读节奏。

本文为本轮新增待确认补全内容，用户确认前不得直接作为正文正史。

## 总体节奏原则

第一阶段不能写成主角一路赢。读者追读点应当来自“他明明看见了关键，却总差一点被世界承认”。

第一阶段读者情绪曲线：

低位憋屈 -> 小亮光 -> 有限承认 -> 名声反噬 -> 崩盘代价 -> 放弃证明 -> 低位入场。

## 单元验收

第一阶段每十章检查一次：

1. 至少一次爽点必须被代价抵消。
2. 至少一次别人反对主角是有道理的。
3. 至少一次主角表达伤人后不能立刻修复。
4. 至少一次赵倩倩的稳定路线比主角更可信。
5. 每个单元章末都要让读者问“下一步怎么办”，不能只问“主角什么时候赢”。
