# 第一阶段伏笔闭环表

## 实现原理

本文件补足第一阶段六十章的伏笔闭环管理。它从 [[剧情大纲/定稿/第一阶段白丁境/第一阶段逐章场景卡|第一阶段逐章场景卡]] 和 [[剧情大纲/定稿/全局/伏笔章节投放表|伏笔章节投放表]] 中抽出第一阶段必须投放、推进、阶段内回收或长线延迟回收的内容。

## 如何使用

写第一阶段正文、复盘十章或调整场景卡时，用本表检查伏笔是否只投不收、是否突然救场、是否越过代价。

本文为本轮新增待确认补全内容，用户确认前不得直接作为正文正史。

## 十章复盘检查

每十章复盘时，必须回答：

1. 本单元是否至少投放一个后续会回收的伏笔。
2. 本单元是否至少推进一个此前已投伏笔。
3. 本单元是否有一个伏笔带来代价，而不是只带来奖励。
4. 本单元是否有一个别人反对主角的理由在后文被证明有价值。
5. 本单元是否避免把伏笔写成外挂、后台或巧合救场。

## 阶段出口闭环

第一阶段结束前必须完成以下闭环：

1. 仪器误差从偶然观察变成组织性瑕疵入口。
2. 旧护具从穷办法变成低成本救援价值。
3. 表达刺人从缺点变成必须训练的执行能力。
4. 救援代价从小伤提醒变成名额争议核心。
5. 军方评语从观察变成第二阶段主线压力。
6. 赵倩倩线从少年吸引推进到现实裂缝。
