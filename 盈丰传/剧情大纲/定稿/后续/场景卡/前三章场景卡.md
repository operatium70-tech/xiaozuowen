# 前三章场景卡

## 实现原理

本文件把开局十章中的前三章拆成可写作的场景卡，只定义场景冲突、视角、转折和章末钩子，不写小说正文。

## 如何使用

写前三章正文前，先读 [[系统规则/开写白名单|开写白名单]]、[[剧情大纲/定稿/后续/十章细纲/赛前铺垫与系内选拔起势细纲|赛前铺垫与系内选拔起势细纲]] 和本文件。

本文为已确认，不能直接当作正文。

| 项目 | 内容 |
| --- | --- |
| 章节功能 | 开局立 [[人物设定/赵盈丰|赵盈丰]] 的处境：聪明、刺人、低存在感、想往上爬 |
| 核心场景 | 资源鉴定冷门课程或实验课，标准检测仪给出一个看似稳定的结论 |
| 视角重心 | 赵盈丰观察仪器误差、样本边缘变化和环境扰动 |
| 表层冲突 | 他指出异常，但表达方式刺人，让同学和助教不舒服 |
| 深层冲突 | 他说的是对的，但他还不会让别人愿意听 |
| 关键人物 | [[人物设定/赵盈丰|赵盈丰]]、[[人物设定/学院配角/陈拙|陈拙]] |
| 转折 | 陈拙不奖励他，只要求他写出样本、误差、验证步骤 |
| 章末钩子 | 赵盈丰看到学院比赛初筛通知，发现资源鉴定冷门项也被纳入计分 |
| 必须呈现 | 赵盈丰的细心、长期积累、被嫌弃、仍然不服输 |
| 禁止写法 | 不写成全场震惊，不写成导师当场收徒，不写成天才鉴宝 |

## 第三章场景卡

| 项目 | 内容 |
| --- | --- |
| 章节功能 | 建立 [[人物设定/赵倩倩|赵倩倩]] 对照线，让读者看见她真的优秀 |
| 核心场景 | 团队项目、基础训练或小型模拟任务中，赵倩倩稳定完成协作 |
| 视角重心 | 赵盈丰既被她吸引，也被她的稳定刺痛 |
| 表层冲突 | 赵盈丰发现自己的奇招很亮，但赵倩倩的正统路线更容易被组织信任 |
| 深层冲突 | 嫡系优势不是只靠资源，而是训练体系、情绪稳定和团队语言 |
| 关键人物 | [[人物设定/赵倩倩|赵倩倩]]、[[人物设定/赵盈丰|赵盈丰]]、[[人物设定/学院配角/林照夜|林照夜]] |
| 转折 | 赵盈丰第一次承认：赵倩倩不是只靠出身，她是真的稳 |
| 章末钩子 | 两人因比赛报名或小组调配产生直接交集 |
| 必须呈现 | 真心吸引、阶层刺痛、同族同学关系、少年情愫的起点 |
| 禁止写法 | 不写成恋爱脑，不让赵倩倩崇拜赵盈丰，不提前制造狗血误会 |

## 三章整体目标

前三章要让读者记住三件事：

1. 赵盈丰有光，但这道光会刺伤别人和自己。
2. 赵倩倩不是奖励，而是稳定、正统、优秀和阶层轨迹的对照。
3. 学院比赛不是拿奖杯，而是普通学生进入更大规则的窗口。
