# 阶段功能定义

## 实现原理

本文件把第二阶段的修炼功能、社会功能、人物功能和剧情功能分开，防止后续把 [[世界观/炼气境|炼气境]] 写成单纯升级打怪。

## 如何使用

拆第二阶段事件时，先判断该事件承担哪一种功能。若一个事件只提供爽点、不改变赵盈丰的方法、身份或关系，就不应保留。

## 修炼功能

炼气境的修炼功能是延长上升期，而不是战力碾压。

本阶段应重点展示：

1. 赵盈丰通过氧元素、代谢调控、医学修复和元能循环，让身体维持更长训练周期。
2. 他比白丁境更能恢复、更能熬、更能稳定使用工具和装备。
3. 他能把元素认知转化为旧装备维护、药剂判断、低成本材料组合和环境风险识别。
4. 他仍不能肉身硬抗重武器、战甲、飞梭武器和高阶修士。

## 社会功能

第二阶段要让赵盈丰看见：成绩只是入口，制度承认才是通道。

学院间联合赛、荒区实习和地方任务都不是单纯竞技，而是军方、商会、学院、家族、地方机构共同观察人的筛选系统。

赵盈丰必须学会：

1. 谁有建议权。
2. 谁有执行权。
3. 谁承担事故责任。
4. 谁定义任务成功。
5. 谁掌握事后评价。

## 人物功能

第二阶段不急着大量新增人物，优先复用第一阶段已建立的功能网。

| 功能 | 可承接人物或组织 | 作用 |

| 证据链训练 | [[人物设定/学院配角/陈拙|陈拙]]、学院评审组 | 要求赵盈丰把判断写成可复核报告 |
| 纪律与执行 | [[人物设定/学院配角/林照夜|林照夜]]、联合赛队伍 | 逼他学会在队伍规则内表达 |

## 剧情功能

本阶段要完成四次升级：

1. 从替补到稳定贡献者。
2. 从口头判断到证据链报告。
3. 从临场奇招到可执行方案。
4. 从个人正确到组织采纳。

## 失败功能

赵盈丰必须继续失败。

建议至少安排三类失败：

1. 证据不足：判断方向对，但样本不足，无法说服评审。
2. 权责错位：方案可行，但他没有执行权，强推导致队伍反感。
3. 结果被抢：低成本方案有效，但被商会、队伍或家族包装成别人的成果。

这些失败不能只用于虐主，而要逼他升级方法。

## 情感功能

第二阶段的感情线重点不是甜或虐，而是现实压力逐步显影。

赵盈丰与 [[人物设定/赵倩倩|赵倩倩]] 的差距应体现在：

1. 她更容易被安排进稳定高阶路线。
2. 她的家庭能提前参与未来规划。
3. 她的失败会被系统托住，赵盈丰的失败会直接影响入口资格。
4. 两人不是不真心，而是越来越难共享同一套节奏。

## 阶段终点功能

第二阶段终点应让赵盈丰获得一次低位但真实的社会承认。

建议出口为：母星级外勤观察资格、联合项目边缘名单、荒区技术助理候选或军方技术观察名单。

出口必须保留压力：他仍不是核心，只是终于进入更真实的母星级任务系统。
