# 第二阶段炼气境

## 实现原理

第二阶段承接第一阶段的替补名额，把 [[赵盈丰]] 从学院内部公开选拔推入更真实的联合赛、荒区实习和地方任务。

本阶段的核心不是让赵盈丰变成强者，而是让他明白：进入 [[世界观/炼气境|炼气境]] 后，能干预自身寿命、维持身体上升期、驾驭工具和装备，但仍然不能靠战力碾压世界。

## 阶段定位

第一阶段让赵盈丰被看见。

第二阶段让赵盈丰学会让组织相信他。

他从“能发现异常”升级到“能把异常变成证据链、方案、分工和责任边界”。这一步比赢比赛更难，因为真实任务不会因他判断正确就自动转向。

## 大舞台

第二阶段建议由三段构成：

1. 学院间联合赛：承接第一阶段替补名额，让赵盈丰以低位身份进入更大舞台。
2. 荒区实习：把比赛规则换成半真实风险，放大救援、装备、环境、队伍纪律和组织响应。
3. 地方任务：让学院成绩进入社会评估，连接军方、商会、家族、地方机构和赵氏重评。

## 核心矛盾

赵盈丰进入炼气境后，优势变成更长训练周期、更好恢复、更稳定工具使用、更强材料和装备转化能力。

但他的旧问题仍在：

1. 他能看见问题，但别人未必愿意听。
2. 他能提出方案，但组织未必愿意承担风险。
3. 他能低成本改造资源，但商业和制度会争夺解释权。
4. 他能压住情绪一时，但面对身份轻视和赵倩倩差距时仍会被刺痛。
5. 他能借规则，但还不会参与制定规则。

## 赵盈丰阶段变化

| 阶段位置 | 旧状态 | 新训练目标 |
| --- | --- | --- |
| 联合赛前段 | 替补、低存在感、被动证明 | 学会在低位身份下稳定贡献 |
| 联合赛中段 | 判断锋利，但沟通成本高 | 学会把判断写成证据链和执行方案 |
| 荒区实习 | 想用正确判断扭转局势 | 学会区分建议权、执行权和责任权 |
| 地方任务 | 想靠结果证明自己 | 学会让结果被制度承认 |
| 阶段尾声 | 被看见的旁支学生 | 获得母星级外勤观察资格或正式职业入口 |

## 赵倩倩线

[[赵倩倩]] 在第二阶段不应退出，也不应变成单纯恋爱压力。

她的功能是继续作为稳定、正统、嫡系平台和家庭预期的对照。她与赵盈丰都在上升，但进入的队伍、导师、资源、家庭会议和未来安排逐渐不同。

本阶段可以开始显露“有缘无分”的结构压力，但不得把她写成嫌贫爱富，也不得把赵家嫡系写成恶毒反派。

## 阶段出口

第二阶段结尾不建议给赵盈丰冠军、正式核心或一战成名。

更合适的出口是：他因一次荒区或地方任务中的证据链、救援协同和低成本资源重组，获得母星级外勤观察资格，或进入军方、商会、学院联合项目的边缘名单。

这仍然只是下一层最低入场券。

## 禁止写法

1. 不把炼气境写成超人战力境界。
2. 不让赵盈丰凭战力压服联合赛和荒区任务。
3. 不让组织无条件相信赵盈丰。
4. 不让导师、军方、商会或赵氏家族给他无代价捷径。
5. 不确认最终女主。
6. 不把赵倩倩线提前写成决裂或反派打压。
