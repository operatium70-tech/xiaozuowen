# 剧情大纲

## 实现原理

本目录只保留可服务后续写作的定稿大纲、复盘、时间线和后续细纲。

## 如何使用

开写正文前，只把 `定稿` 作为直接依据；待确认问题不能写成定稿。

## 目录结构

```text

├── README.md
├── 待确认问题.md
└── 定稿/

    ├── 复盘/
    │   ├── 总览.md
    │   └── 十章复盘/
    ├── 时间线/
    │   ├── 总览.md
    │   └── 十章时间线/
    └── 后续/
        └── 十章细纲/
```
