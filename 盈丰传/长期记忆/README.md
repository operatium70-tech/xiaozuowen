# 长期记忆

## 实现原理

长期记忆记录当前创作状态、剧情状态、时间线、伏笔、人物关系、世界变化和统计，解决长篇上下文丢失问题。

## 如何使用

每次继续项目先读当前状态，再读已确认编号记录和设定归档。

## 推荐读取顺序

1. [[当前状态/当前创作目标]]
2. [[当前状态/当前剧情状态]]
3. [[当前状态/最近章节总结]]
4. [[当前状态/当前时间线]]
5. [[当前状态/当前伏笔状态]]
6. [[当前状态/当前战力状态]]
7. [[当前状态/当前人物关系]]
8. [[当前状态/当前世界变化]]
9. [[当前状态/字数统计]]
10. [[当前状态/已发布边界]]
11. [[已确认编号记录]]
12. [[设定归档]]
