# 白丁境

## 实现原理

白丁境不是修士，只是接受修行教育的普通人，用来承载 [[赵盈丰]] 长期求学、积累、试错、复盘的蓄水阶段。

## 如何使用

第一阶段写作或大纲整理中，白丁境不得表现为真正主动逆熵修行，也不得拥有超人战力。

## 最终定义

白丁境不是修士，只是接受修行教育的普通人。

白丁境和普通人的区别很小，核心区别是正在接受系统学习、基础训练和生活技能教育，但身体本质仍然受自然衰老支配。

## 核心内容

1. 元素物质认知：知道 118 种元素，知道常见物质形态。
2. 微观基础物理认知：知道物质拆解层级为物质 -> 分子 -> 原子 -> 质子 / 中子 / 电子 -> 夸克 -> 标准模型基本粒子。
3. 技能生活化：具备基本生活技能，可以通过知识、工具、训练和基础仪器改善生活，但不能真正主动干预寿命。

## 寿命逻辑

30 岁以后身体开始逐步走下坡，恢复力、精力、代谢、反应都会自然衰退。
