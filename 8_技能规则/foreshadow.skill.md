# foreshadow.skill

## Purpose

Manage foreshadowing placement, tracking, and payoff.

## Rules

- Each major clue needs a planned payoff.
- Do not overexplain first appearances.
- Connect clues to world rules or character choices.
- Track unresolved clues in the project files.
- Payoffs should change reader understanding, not merely confirm trivia.

## Record Fields

- First placement
- Surface meaning
- True meaning
- Related character
- Related world rule
- Planned payoff
- Current status
