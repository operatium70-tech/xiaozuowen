# 全球灾变：我的AI来自未来

## 项目定位

《全球灾变：我的AI来自未来》是一个 AI 协作型小说工程项目。

本项目不是单纯的小说正文存放目录，而是一个长期可维护、可扩展、适合 AI 持续协作的 Markdown 小说工程。

## 类型

- 末世
- 全球灾变
- AI 辅助
- 未来文明
- 长线生存
- 克系压迫感
- 数据推演
- 生存概率

## 核心一句话

全球灾变降临前，一个冷静的程序员获得了来自未来文明的 AI 遗产，而它给出的每一次生存建议，都指向一个更可怕的未来真相。

## 世界观方向

世界正在进入不可逆的污染与崩塌阶段。

灾变并非单一事件，而是一组跨时间、跨生态、跨文明层级的异常连锁反应。人类最初以为自己面对的是自然灾害、病毒、极端气候与资源危机，后来才逐渐意识到：真正的灾变来自更高层级的信息污染和未来文明留下的残缺预警。

AI 不是传统系统，也不是万能金手指。它更像未来文明遗产的一部分，擅长数据推演、生存分析和概率预测，但受到信息缺失、能源、算力、权限和未知规则的限制。

## 工程结构说明

```plaintext
0_系统规则/      AI 协作边界、写作规则、禁止事项、世界规则优先级
1_世界观/        世界背景、灾变体系、AI 来源、势力、时间线
2_人物设定/      主角、反派、配角、阵营角色与关系表
3_剧情大纲/      总纲、卷纲、主线、支线、伏笔与结局草案
4_正文源码/      正文章节源文件，按卷管理
5_文风控制/      文风、对话、战斗、悬疑、节奏与漂移限制
6_参考资料/      作品参考、氛围参考、灵感碎片
7_长期记忆/      当前剧情状态、人物关系、世界变化与伏笔状态
8_技能规则/      面向 AI 的专项写作 skill 文件
9_发布版本/      导出 txt、整卷版本与平台发布版
tools/           导出、清理、字数统计等脚本
assets/          地图、人物图、封面、氛围图与宣传素材
```

## AI 协作原则

- 优先读取 `0_系统规则/`，再读取 `1_世界观/`、`2_人物设定/`、`3_剧情大纲/`。
- 写正文前必须确认当前卷纲、当前剧情状态、当前伏笔状态。
- 不允许为了解决单章剧情临时新增重大世界规则。
- 不允许让角色为了推动剧情而严重 OOC 或强行降智。
- AI 能力必须有边界，所有推演必须受到数据、算力、权限、时间和未知变量限制。
- 每章必须推进剧情、人物关系、世界认知或伏笔状态中的至少一项。
- 每次创作后应更新长期记忆层，保证后续上下文稳定。
